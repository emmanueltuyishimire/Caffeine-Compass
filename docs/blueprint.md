# **App Name**: Caffeine Compass

## Core Features:

- Intake Calculation: Calculate total caffeine consumption from various drinks using a searchable database and user input.
- Half-Life Estimation: Estimate how long caffeine remains active in the body, showing a visual decay chart based on individual factors.
- Sleep Impact Prediction: Predict the risk of caffeine affecting sleep based on consumption timing and amount.
- Personalized Caffeine Assistant: A Gemini AI-powered tool answering specific caffeine-related questions, providing insights from calculators and the database.
- Drink Comparison Database: Searchable database providing caffeine information (mg per serving) of common drinks. This will be stored in Firestore.
- Sensitivity Estimation: Assess user sensitivity to caffeine. This tool leverages stored data within Firestore.
- Withdrawal Management: Allow the users to minimize withdrawal headaches. Data should be stored and easily accesible using Firestore.

## Style Guidelines:

- Primary color: Coffee brown (#6F4E37) to evoke a sense of warmth and comfort related to coffee.
- Background color: Off-white (#FAF9F6), nearly the same hue as the primary color but desaturated to 12% and brightened, to provide a calm and readable surface.
- Accent color: Mint green (#3EB489), chosen as an analogous color, to create good contrast and relate to the calming effect of better focus, health and sleep.
- Body and headline font: 'Inter' (sans-serif) for clean readability across the site.
- Use clear, scientific icons representing caffeine, sleep, and health.
- Maintain a clean, minimal UI with high contrast for easy data visualization.
- Subtle animations for data loading and transitions.
